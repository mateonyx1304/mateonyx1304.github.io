🎨 Linktree Personal - Mateo Angel

📌 Descripción

---

Página tipo Linktree desarrollada con HTML y CSS, diseñada para centralizar enlaces personales en una interfaz visual atractiva. Incluye animaciones, fondo dinámico y estilos personalizados enfocados en mejorar la experiencia del usuario.

---

👨‍💻 Enfoque del proyecto

Este proyecto está orientado a la práctica de:

Animaciones CSS

Diseño visual y experiencia de usuario (UI/UX básica)

Integración de estilos personalizados con Bootstrap

Uso de imágenes dinámicas (avatar de GitHub)

---

🛠️ Tecnologías utilizadas

HTML5

CSS3 (animaciones, gradients, keyframes)

Bootstrap 4

Git y GitHub

---

⚙️ Características principales

---

Fondo animado con gradiente dinámico

Animación personalizada (@keyframes)

Diseño centrado tipo Linktree

Integración de avatar dinámico desde GitHub

Botones interactivos con enlaces externos

Uso de sombras y bordes redondeados para mejorar UI

---

🎯 Objetivo

Construir una página simple pero visualmente atractiva que permita practicar animaciones y estilos avanzados en CSS, además de servir como punto de acceso a mis perfiles digitales.
